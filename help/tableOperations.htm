Generic table tools

This set of tools allows you to perform operations on data. These operations are applied to the data already loaded in TAPIS. These tools can also be used with STA data, but you must take into account that from this point you cannot send another request to the STA service and the data will become table-based data.

The operations are applied manually (implemented in the TAPIS code) without sending external requests. You may find some tools that are similar to the STA tools, but in this case the operations are applied to data that has already been loaded.

With these tools you can select, filter, and edit data, create new columns, group data, and perform other operations.

Create Columns

Setting Up the Node:

How to Use the Node:

Create Columns tool allows you to create new columns in your table. You can create three kinds of columns:

  • Column with a constant value
  • Column with an auto-incremental value starting from an initial value
  • Empty column (you can edit it record by record using the Edit record tool)
This tool can be used with data from your computer (table data) or with a table requested from an internet service. Remember that once you use this tool, you cannot make another request to the server again.

For the example shown, CSV import has been used.

This is the original data, containing only the columns shown in the image below.

General overview

To display the dialog, double-click on the node. This dialog has two parts: the upper part, where you can configure the column you want to add (Select operation, Column name, Add column to list button), and the List of columns to be added (columns that you have configured to be added to your table).

Step 1: Choose the type of column you want to add and fill in the fields if it will be a constant value or an auto-incremental value. If you leave both values empty, it will be considered as 0.

Step 2: Enter a name for the new column. It cannot be empty.

Step 3: Click the Add column to list button to add this configuration to the table above. The program only creates the columns that appear in this table.

The creation of a Constant value column is used to illustrate the previous steps. You can add any type of value as a constant.

Remove a column from the list: To remove a column, click the "trash" button and the row will disappear.

To create an auto-incremental column, follow the previous steps and provide an initial value. This value increases by 1 for each record.

To create an empty column, follow the same steps. It is only necessary to provide a column name.

In the image above, all three columns have been added. You can add as many columns as you want each time you use the tool (in a new node).

Save Table

Setting Up the Node:

How to Use the Node:

Save Table tool allows you to save node data as a CSV file and attributes as CSVW (JSON format). You can save any kind of data.

In the example below, the data is saved after applying a filter, so only the filtered data will be saved.

To display the dialog, double-click on the node. In the dialog, you can perform two actions:

Save CSV: Save node data as a CSV file, choosing the delimiter you want to use (enter it in the CSV delimiter field).

Save CSVW: Save attributes from the data. The resulting JSON file will only contain the properties of completed attributes.

If you want to save both data and attributes, you will need to perform both actions.

Click the button for the desired action, enter a name for the document, and save it.
