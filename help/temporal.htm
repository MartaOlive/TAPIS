As with any entity, to view the Campaigns of an STA service, this tool must be connected directly to the STA service or to a related entity following the STA schema. If it is connected to another entity, that entity must contain only a single record. To achieve this, you can use the Select row, Filter rows, or Select resource tool.

Double left-click on the node to open the dialog. By default, 100 records are retrieved from the STA service. Use this dialog to specify a higher number of records.
