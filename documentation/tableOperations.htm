Generic table tools

This set of tools allows you to perform operations on data. These operations are applied to the data already loaded in TAPIS. These tools can also be used with STA data, but you must take into account that from this point you cannot send another request to the STA service and the data will become table-based data.

The operations are applied manually (implemented in the TAPIS code) without sending external requests. You may find some tools that are similar to the STA tools, but in this case the operations are applied to data that has already been loaded.

With these tools you can select, filter, and edit data, create new columns, group data, and perform other operations.
