STA entities reading tool

This set of tools is used to explore entities in an STA service, so they can only be linked to a node containing STA data. You can only link an entity with a related entity.(To know which entities are linked, consult the schema below)

Any entity can be linked to an STA service node. If you want to explore an entity related to your previous entity, you must ensure that your data contains only one record (by applying the Select tool, the Filter tool, or automatically clicking the entity in the table below).

Each time you link an entity,a new request to the API will be made . Every request to the STA service will return 100 records. Double-clicking on this node will display a window where you can enter a number to specify how many records you want to load.
